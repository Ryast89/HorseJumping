The game is "won" when the player reaches a certain score, in this case 2000. 
If you'd like to change that score to make it easier to reach, you can do that on line 116 by changing the "10000"
to an easier number. (That number is divided by 5 to make the visible score)

The game file that should be run is horsejumping.html. The other files are used in the iframe.
